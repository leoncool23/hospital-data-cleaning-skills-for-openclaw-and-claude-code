# 编码标准与单位换算映射表

## ICD-10 格式校验

```regex
有效的 ICD-10-CM 格式：^[A-Z]\d{2}(\.\d{1,4})?$
正确示例：J18.9   A41.9   I21.0   Z87.891
错误示例：j18.9（小写字母）  18.9（缺少字母）  J18..9（双点）
```

## ICD 版本对照

| 版本 | 时间段 | 处理方式 |
|------|--------|---------|
| ICD-9-CM | 2016年前 | 使用 GEM 对照文件（CMS 通用等价映射）转换 |
| ICD-10-CM | 2016年至今（美国） | 当前标准 |
| ICD-10（WHO） | 国际通用 | 跨国比较时使用 |
| ICD-11 | 2022年起（WHO） | 核实源系统是否已使用 ICD-11 |

## 剂量单位换算 → mg（目标单位）

| 来源单位 | 换算系数 → mg |
|---------|-------------|
| g | × 1000 |
| mg | × 1 |
| μg / mcg | × 0.001 |
| ng | × 0.000001 |
| mEq | 依赖上下文（见电解质分子量表） |
| IU | 药品特异性（见下方IU换算表） |

## IU → 质量换算（常见药品）

| 药品 | 1 IU 等于 |
|------|----------|
| 胰岛素 | 0.0347 mg |
| 肝素 | ≈ 0.002 mg |
| 维生素A | 0.0003 mg |
| 维生素D | 0.025 μg（= 0.000025 mg） |
| 青霉素G | 0.0006 mg |

## 用药频次代码标准

| 代码 | 含义 | 每日次数 |
|------|------|---------|
| QD / qd | 每日一次 | 1 |
| BID / bid | 每日两次 | 2 |
| TID / tid | 每日三次 | 3 |
| QID / qid | 每日四次 | 4 |
| Q6H | 每6小时一次 | 4 |
| Q8H | 每8小时一次 | 3 |
| Q12H | 每12小时一次 | 2 |
| PRN / prn | 需要时 | 不定 |
| st. / STAT | 立即/单次 | 1 |
| qn | 每晚一次 | 1 |
